\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classlinktuit_1_1linktuit_1_1modal_1_1_url_mapping}{Url\+Mapping}} class represents the entries that will be saved in the database. Each entry will have a unique ID that is supplied by the database, the original long U\+RL, and the number of requests left until the entry expires.
